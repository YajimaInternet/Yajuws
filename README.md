## ビルドと実行 (v4.1)

```bash
go build -o yajuws .
```

Windows (PowerShell / cmd):

```powershell
.\yajuws.exe
```

macOS / Linux:

```bash
./yajuws
```

- 便利ツール (時計/タイマー/稼働時間/じゃんけん)

- テーマ/カラー/高速起動の設定
- タスクマネージャー (CPU使用率/プロセス数/メモリ)
